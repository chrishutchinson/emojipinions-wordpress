Add emoji reactions to posts and pages.

== Description ==

Allows the content editor to set a list of emoji that readers can click to 'react' to. Totals can be shown below each emoji. Data will be stored against the post, page, or custom post type as custom fields.

== Installation ==

1. Upload the `emojipinions` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
