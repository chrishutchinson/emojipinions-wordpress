Add emoji reactions to your WordPress posts.

== Description ==

Add emoji reactions to your WordPress posts! The emojis that readers can react with can be configured on each post in the editor. Totals can be shown below each emoji. Data will be stored against the post, page, or custom post type as custom fields.

== Installation ==

1. Upload the `emojipinions` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
